# BaseResourceToSpreadsheet

Een simpel Python-scriptje om een FHIR base-resource te converteren naar een Excel-spreadsheet. Het formaat is gebaseerd op https://confluence.hl7.org/spaces/FHIR/pages/35718548/FHIR+Spreadsheet+Authoring, maar niet alles hiervan wordt meegenomen in de output.

De default actie van het script is om alle R4-resources als spreadsheet uit te spugen.
